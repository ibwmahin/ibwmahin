## How the automation works (technical summary)
- A GitHub Action triggers a Node.js script on `push` and daily schedule.
- The script pulls GitHub data (commits, languages, contributions), generates charts using QuickChart or Chart.js, saves PNG/SVG to `assets/charts/`, and commits them back to the repo.
- Uses `GITHUB_TOKEN` to commit; no extra secrets required.
